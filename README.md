# image_filter_black


## Prepare environment:

    git clone https://github.com/vilnic/image_filter_black
    cd image_filter_black
    python3 -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt

## run script:

    source .venv/bin/activate
    python image_filter_black.py -i <image>
